GA4 ANALYTICS ALL SITES
這裡看的是「有多少人來你的網站、待多久、看了幾頁」。

Active Users — 有多少不同的人來過（同一個人來 3 次只算 1 個）
Sessions — 總共被造訪幾次（同一個人來 3 次就算 3 次）
Page Views — 被看了幾個頁面（一個人看 5 頁就算 5）
Avg Duration — 平均每次來停留多久（越久越好，代表內容有吸引力）
Bounce Rate — 只看一頁就離開的比例（越低越好，低於 50% 算不錯）

REALTIME — 此刻正在你網站上的人，即時更新。

★ 小建議：Bounce Rate 超過 70% → 首頁可能不夠吸引人，試試加 CTA 或調整排版
★ 平均停留低於 30 秒 → 內容可能太短或不符合搜尋意圖
★ Page Views / Sessions 比值大於 2 → 很棒，訪客願意多看幾頁
